Qual a probabilidade de P(A U B)? 
{
= 5/6
~ 1/6

}

Qual o enunciado correto do Teorema de Bayes? ($\cap$ \= intersecção, $\cup$ \= união)
{
= P(A|B) \= P(A|B) P(A) / P(B)
~ P(A $\cap$ B) \= P(A|B) P(B)
~ P(A $\cap$ B) \= P(A) P(B)
~ P(A $\cup$ B) \= P(A|B) P(B)
~ P(A $\cup$ B) \= P(A|B) P(A)
}

Qual expressão melhor define A e B como eventos independentes? ($\cap$ \= intersecção, $\cup$ \= união)
{
~ P(A|B) \= P(A|B) P(A) / P(B)
= P(A $\cap$ B) \= P(A|B) P(B)
~ P(A $\cap$ B) \= P(A) P(B)
~ P(A $\cup$ B) \= P(A|B) P(B)
~ P(A $\cup$ B) \= P(A|B) P(A)
}

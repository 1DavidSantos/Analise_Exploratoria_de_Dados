\n Exercício 3.1. Por quê o gráfico aparentemente não exibe os quartis de cada data?
{
= Por que há um único valor para cada data e, portanto, dos os quartis coincidem
~ Devido a falta de precisão do gráfico
~ Os tipos dos valores não são contínuos
~ Os tipos dos valores não são discretos 
~ Nenhuma das alternativas
} 

Associe corretamente o gráfico mais característico\:
{ 
= Evolução -> gráficos de linha
= Distribuição -> histogramas
= Composição -> gráficos de barras
= Relações -> gráficos de dispersão
} 

O Boxplot é um gráfico que permite principalmente entender\:
{ 
= A distribuição dos dados
~ A composição de diferentes dados 
~ As relações entre diferentes variáveis
~ A evolução de uma variável aleatória
~ A dispersão de uma variável com relação a outra
}
